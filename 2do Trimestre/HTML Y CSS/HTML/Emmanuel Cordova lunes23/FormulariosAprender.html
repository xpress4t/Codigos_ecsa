Explicación

Etiqueta form: etiqueta para formularios

Atributo name: sirve para asignar un nombre al elemento

action: para enviar el formulario a un archivo (por ejemplo)

get y post: formas de enviar el contenido (información) del formularios. Con el get, la información se envía por la URL y es visible. Bajo ningún concepto se utiliza el método get cuando se envíe una contraseña

Etiqueta label: para poner un texto "asociado" al elemento de formulario

Atributo type: para indicar el tipo de input (text, number, email, password, search, submit, reset, etc.)

autofocus: posiciona el cursor dentro del elemento

submit: es para enviar el formulario

value: asigna un valor al elemento

minlength y maxlength: número mínimo y máximo de caracteres de un campo (elemento)

number: para introducir números enteros, salvo que utilice también el atributo step, en cuyo caso permitiría decimales.El atributo step nos sirve para enviar números decimales (0.001 permitiría enviar milésimas)

min y max: rango mínimo y máximo para datos numéricos

placeholder: sirve para aportar información que desaparece al escribir dentro del elemento

radio: para seleccionar marcando una opción (excluyente). Pon el mismo name para todos
